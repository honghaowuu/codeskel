# codeskel Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `codeskel`, a fast Rust CLI that scans a project, builds a dependency graph via tree-sitter, extracts signatures, measures docstring coverage, and writes a cache for LLM-driven comment generation.

**Architecture:** Three commands (`scan`, `get`, `rescan`) share a JSON cache at `.codeskel/cache.json`. Parsing is tree-sitter based for 9 languages; files are processed in parallel with `rayon`; dependency ordering uses Kahn's topological sort with cycle breaking.

**Tech Stack:** Rust, Clap 4 (derive), tree-sitter 0.22 + grammar crates, rayon, serde/serde_json, walkdir, globset, chrono, anyhow.

```rust
/// Returns true if the file appears to be auto-generated.
/// Checks: first 10 lines for known markers, path patterns.
pub fn is_generated(rel_path: &str, content: &str) -> bool {
    if path_looks_generated(rel_path) {
        return true;
    }
    content_looks_generated(content)
}

fn path_looks_generated(rel_path: &str) -> bool {
    let patterns = [
        "/generated/",
        "_pb.java",
        ".g.ts",
        "/migrations/",
        "_gen.go",
    ];
    patterns.iter().any(|p| rel_path.contains(p))
}

fn content_looks_generated(content: &str) -> bool {
    let markers = [
        "@Generated",
        "// Code generated",
        "DO NOT EDIT",
        "Auto-generated",
        "THIS FILE IS AUTOMATICALLY GENERATED",
    ];
    content
        .lines()
        .take(10)
        .any(|line| markers.iter().any(|m| line.contains(m)))
}
```

```rust
use crate::lang::{detect_language, lang_from_str};
use crate::models::Language;
use globset::{Glob, GlobSet, GlobSetBuilder};
use std::path::{Path, PathBuf};
use walkdir::WalkDir;

pub struct WalkConfig {
    pub forced_lang: Option<Language>,
    pub include_globs: Vec<String>,
    pub exclude_globs: Vec<String>,
}

const DEFAULT_EXCLUDES: &[&str] = &[
    "**/test/**",
    "**/tests/**",
    "**/*Test.java",
    "**/node_modules/**",
    "**/vendor/**",
    "**/.git/**",
    "**/target/**",
    "**/build/**",
    "**/dist/**",
];

pub struct DiscoveredFile {
    pub abs_path: PathBuf,
    pub rel_path: String,
    pub language: Language,
}

pub fn walk_project(root: &Path, cfg: &WalkConfig) -> anyhow::Result<Vec<DiscoveredFile>> {
    let exclude_set = build_glob_set(&cfg.exclude_globs, DEFAULT_EXCLUDES)?;
    let include_set = if cfg.include_globs.is_empty() {
        None
    } else {
        Some(build_glob_set(&cfg.include_globs, &[])?)
    };

    let mut files = Vec::new();
    for entry in WalkDir::new(root).into_iter().filter_map(|e| e.ok()) {
        if !entry.file_type().is_file() {
            continue;
        }
        let abs = entry.path().to_path_buf();
        let rel = abs.strip_prefix(root)
            .map(|p| p.to_string_lossy().into_owned())
            .unwrap_or_default();

        if exclude_set.is_match(&rel) {
            continue;
        }
        if let Some(ref inc) = include_set {
            if !inc.is_match(&rel) {
                continue;
            }
        }

        let lang = if let Some(ref l) = cfg.forced_lang {
            l.clone()
        } else if let Some(l) = detect_language(&abs) {
            l
        } else {
            continue;
        };

        files.push(DiscoveredFile { abs_path: abs, rel_path: rel, language: lang });
    }
    Ok(files)
}

fn build_glob_set(user: &[String], defaults: &[&str]) -> anyhow::Result<GlobSet> {
    let mut builder = GlobSetBuilder::new();
    for g in defaults.iter().map(|s| s.to_string()).chain(user.iter().cloned()) {
        builder.add(Glob::new(&g)?);
    }
    Ok(builder.build()?)
}
```

```rust
pub fn write_cache(cache_dir: &Path, cache: &CacheFile) -> anyhow::Result<()> {
    std::fs::create_dir_all(cache_dir)?;
    let path = cache_dir.join("cache.json");
    let json = serde_json::to_string_pretty(cache)?;
    std::fs::write(&path, json)
        .with_context(|| format!("Cannot write cache to {}", path.display()))?;
    Ok(())
}

pub fn read_cache(cache_path: &Path) -> anyhow::Result<CacheFile> {
    let content = std::fs::read_to_string(cache_path)
        .with_context(|| format!("Cannot read cache from {}", cache_path.display()))?;
    let cache: CacheFile = serde_json::from_str(&content)?;
    Ok(cache)
}
```

```rust
use super::{LanguageParser, ParseResult};
use crate::models::{Param, Signature};
use tree_sitter::{Node, Parser};

pub struct JavaParser;
impl JavaParser { pub fn new() -> Self { Self } }

impl LanguageParser for JavaParser {
    fn parse(&self, source: &str) -> ParseResult {
        let mut parser = Parser::new();
        parser.set_language(tree_sitter_java::language()).expect("Java grammar");
        let tree = match parser.parse(source, None) {
            Some(t) => t,
            None => return ParseResult::default(),
        };
        let root = tree.root_node();
        let bytes = source.as_bytes();

        let mut result = ParseResult::default();
        let mut documented = 0usize;
        let mut total = 0usize;

        walk_java(root, bytes, source, &mut result, &mut documented, &mut total);

        result.coverage = if total > 0 { documented as f64 / total as f64 } else { 1.0 };
        result
    }
}

fn node_text<'a>(node: Node, bytes: &'a [u8]) -> &'a str {
    node.utf8_text(bytes).unwrap_or("")
}

fn preceding_block_comment_is_doc(node: Node, source: &str) -> bool {
    // Walk backwards through prev siblings looking for a block_comment starting with /**
    let mut prev = node.prev_sibling();
    while let Some(p) = prev {
        let kind = p.kind();
        if kind == "block_comment" {
            let text = &source[p.byte_range()];
            return text.starts_with("/**");
        }
        // Skip whitespace/other comments
        if kind == "line_comment" || kind == "\n" {
            prev = p.prev_sibling();
            continue;
        }
        break;
    }
    false
}

fn walk_java(
    node: Node,
    bytes: &[u8],
    source: &str,
    result: &mut ParseResult,
    documented: &mut usize,
    total: &mut usize,
) {
    match node.kind() {
        "package_declaration" => {
            // package com.example; — grab the scoped_identifier
            if let Some(name_node) = node.child_by_field_name("name") {
                result.package = Some(node_text(name_node, bytes).to_string());
            } else {
                // try second child (after 'package' keyword)
                for i in 0..node.child_count() {
                    let c = node.child(i).unwrap();
                    if c.kind() == "scoped_identifier" || c.kind() == "identifier" {
                        result.package = Some(node_text(c, bytes).to_string());
                        break;
                    }
                }
            }
        }
        "import_declaration" => {
            // import java.util.Optional; or import static ...
            for i in 0..node.child_count() {
                let c = node.child(i).unwrap();
                if c.kind() == "scoped_identifier" || c.kind() == "identifier" {
                    result.raw_imports.push(node_text(c, bytes).to_string());
                    break;
                }
            }
        }
        "class_declaration" | "interface_declaration" | "enum_declaration" => {
            let kind = match node.kind() {
                "class_declaration" => "class",
                "interface_declaration" => "interface",
                _ => "enum",
            };
            let name = node.child_by_field_name("name")
                .map(|n| node_text(n, bytes).to_string())
                .unwrap_or_default();
            let has_doc = preceding_block_comment_is_doc(node, source);
            let mut sig = Signature {
                kind: kind.into(),
                name,
                modifiers: extract_modifiers(node, bytes),
                params: None,
                return_type: None,
                throws: vec![],
                extends: None,
                implements: vec![],
                annotations: extract_annotations(node, bytes),
                line: node.start_position().row + 1,
                has_docstring: has_doc,
            };
            // superclass
            if let Some(sc) = node.child_by_field_name("superclass") {
                sig.extends = Some(node_text(sc, bytes).trim_start_matches("extends ").to_string());
            }
            // interfaces
            if let Some(iface) = node.child_by_field_name("interfaces") {
                for i in 0..iface.child_count() {
                    let c = iface.child(i).unwrap();
                    if c.kind() == "type_list" {
                        for j in 0..c.child_count() {
                            let t = c.child(j).unwrap();
                            if t.kind() != "," {
                                sig.implements.push(node_text(t, bytes).to_string());
                            }
                        }
                    }
                }
            }
            *total += 1;
            if has_doc { *documented += 1; }
            result.signatures.push(sig);

            // Recurse into body
            if let Some(body) = node.child_by_field_name("body") {
                walk_java(body, bytes, source, result, documented, total);
            }
            return; // don't double-recurse
        }
        "method_declaration" => {
            let name = node.child_by_field_name("name")
                .map(|n| node_text(n, bytes).to_string())
                .unwrap_or_default();
            let has_doc = preceding_block_comment_is_doc(node, source);
            let return_type = node.child_by_field_name("type")
                .map(|n| node_text(n, bytes).to_string());
            let params = extract_params(node, bytes);
            let throws = extract_throws(node, bytes);
            *total += 1;
            if has_doc { *documented += 1; }
            result.signatures.push(Signature {
                kind: "method".into(),
                name,
                modifiers: extract_modifiers(node, bytes),
                params: Some(params),
                return_type,
                throws,
                extends: None,
                implements: vec![],
                annotations: extract_annotations(node, bytes),
                line: node.start_position().row + 1,
                has_docstring: has_doc,
            });
        }
        "constructor_declaration" => {
            let name = node.child_by_field_name("name")
                .map(|n| node_text(n, bytes).to_string())
                .unwrap_or_default();
            let has_doc = preceding_block_comment_is_doc(node, source);
            *total += 1;
            if has_doc { *documented += 1; }
            result.signatures.push(Signature {
                kind: "constructor".into(),
                name,
                modifiers: extract_modifiers(node, bytes),
                params: Some(extract_params(node, bytes)),
                return_type: None,
                throws: extract_throws(node, bytes),
                extends: None,
                implements: vec![],
                annotations: extract_annotations(node, bytes),
                line: node.start_position().row + 1,
                has_docstring: has_doc,
            });
        }
        "field_declaration" => {
            // Only public/protected fields
            let mods = extract_modifiers(node, bytes);
            if mods.iter().any(|m| m == "public" || m == "protected") {
                let type_ = node.child_by_field_name("type")
                    .map(|n| node_text(n, bytes).to_string())
                    .unwrap_or_default();
                // field declarator name
                let name = node.child_by_field_name("declarator")
                    .and_then(|d| d.child_by_field_name("name"))
                    .map(|n| node_text(n, bytes).to_string())
                    .unwrap_or_default();
                let has_doc = preceding_block_comment_is_doc(node, source);
                *total += 1;
                if has_doc { *documented += 1; }
                result.signatures.push(Signature {
                    kind: "field".into(),
                    name,
                    modifiers: mods,
                    params: None,
                    return_type: Some(type_),
                    throws: vec![],
                    extends: None,
                    implements: vec![],
                    annotations: vec![],
                    line: node.start_position().row + 1,
                    has_docstring: has_doc,
                });
            }
        }
        _ => {
            let mut cursor = node.walk();
            for child in node.children(&mut cursor) {
                walk_java(child, bytes, source, result, documented, total);
            }
        }
    }
}

fn extract_modifiers(node: Node, bytes: &[u8]) -> Vec<String> {
    let mut mods = vec![];
    let mut cursor = node.walk();
    for child in node.children(&mut cursor) {
        match child.kind() {
            "public" | "private" | "protected" | "static" | "final" | "abstract"
            | "synchronized" | "native" | "volatile" | "transient" | "default" => {
                mods.push(child.kind().to_string());
            }
            "modifiers" => {
                let mut c2 = child.walk();
                for m in child.children(&mut c2) {
                    match m.kind() {
                        "public" | "private" | "protected" | "static" | "final" | "abstract"
                        | "synchronized" | "native" | "volatile" | "transient" | "default" => {
                            mods.push(m.kind().to_string());
                        }
                        _ => {}
                    }
                }
            }
            _ => {}
        }
    }
    mods
}

fn extract_annotations(node: Node, bytes: &[u8]) -> Vec<String> {
    let mut anns = vec![];
    let mut cursor = node.walk();
    for child in node.children(&mut cursor) {
        if child.kind() == "modifiers" {
            let mut c2 = child.walk();
            for m in child.children(&mut c2) {
                if m.kind() == "annotation" || m.kind() == "marker_annotation" {
                    anns.push(format!("@{}", node_text(m, bytes).trim_start_matches('@')));
                }
            }
        }
    }
    anns
}

fn extract_params(node: Node, bytes: &[u8]) -> Vec<Param> {
    let mut params = vec![];
    if let Some(fp) = node.child_by_field_name("parameters") {
        let mut cursor = fp.walk();
        for child in fp.children(&mut cursor) {
            if child.kind() == "formal_parameter" {
                let type_ = child.child_by_field_name("type")
                    .map(|n| node_text(n, bytes).to_string())
                    .unwrap_or_default();
                let name = child.child_by_field_name("name")
                    .map(|n| node_text(n, bytes).to_string())
                    .unwrap_or_default();
                params.push(Param { name, type_ });
            }
        }
    }
    params
}

fn extract_throws(node: Node, bytes: &[u8]) -> Vec<String> {
    let mut throws = vec![];
    let mut cursor = node.walk();
    for child in node.children(&mut cursor) {
        if child.kind() == "throws" {
            let mut c2 = child.walk();
            for t in child.children(&mut c2) {
                if t.kind() != "throws" && t.kind() != "," {
                    throws.push(node_text(t, bytes).to_string());
                }
            }
        }
    }
    throws
}
```

```rust
fn first_body_is_docstring(body: Node, bytes: &[u8]) -> bool {
    // body is a `block`; first non-trivial child
    let mut cursor = body.walk();
    for child in body.children(&mut cursor) {
        if child.kind() == "expression_statement" {
            if let Some(inner) = child.child(0) {
                return inner.kind() == "string";
            }
        }
        if !matches!(child.kind(), "\n" | " " | "") {
            break;
        }
    }
    false
}

fn walk_python(
    node: Node,
    bytes: &[u8],
    source: &str,
    result: &mut ParseResult,
    documented: &mut usize,
    total: &mut usize,
    inside_class: bool,
) {
    match node.kind() {
        "import_statement" => {
            let mut cursor = node.walk();
            for child in node.children(&mut cursor) {
                if matches!(child.kind(), "dotted_name" | "identifier") {
                    result.raw_imports.push(node_text(child, bytes).to_string());

            }
        }

                kind: "class".into(),
                name,
                modifiers: vec![],
                params: None,
                return_type: None,
                throws: vec![],
                extends: None,
                implements: vec![],
                annotations: vec![],
                line: node.start_position().row + 1,
                has_docstring: has_doc,
            });

                walk_python(body, bytes, source, result, documented, total, true);
            }
            return;
        }
        "function_definition" => {
            let name = node.child_by_field_name("name")
                .map(|n| node_text(n, bytes).to_string())
                .unwrap_or_default();

            } else {

                    .map(|b| first_body_is_docstring(b, bytes))
                    .unwrap_or(false);

                });
            }
        }
        _ => {
            let mut cursor = node.walk();
            for child in node.children(&mut cursor) {
                walk_python(child, bytes, source, result, documented, total, inside_class);
            }
        }
    }
}
```

```rust
fn is_jsdoc_comment(node: Node, source: &str) -> bool {
    if node.kind() != "comment" { return false; }
    let text = &source[node.byte_range()];
    text.starts_with("/**")
}

fn preceding_jsdoc(node: Node, source: &str) -> bool {
    let mut prev = node.prev_sibling();
    while let Some(p) = prev {
        if p.kind() == "comment" {
            return is_jsdoc_comment(p, source);
        }
        if p.kind() == "\n" { prev = p.prev_sibling(); continue; }
        break;
    }
    false
}

fn walk_ts(node: Node, bytes: &[u8], source: &str, result: &mut ParseResult, documented: &mut usize, total: &mut usize) {
    match node.kind() {
        "import_statement" => {
            if let Some(src) = node.child_by_field_name("source") {
                let raw = node_text(src, bytes).trim_matches('"').trim_matches('\'');
                if raw.starts_with("./") || raw.starts_with("../") {
                    result.raw_imports.push(raw.to_string());
                }
            }
        }
        "class_declaration" | "abstract_class_declaration" => {
            let name = node.child_by_field_name("name")
                .map(|n| node_text(n, bytes).to_string())
                .unwrap_or_default();
            let has_doc = preceding_jsdoc(node, source);
            *total += 1; if has_doc { *documented += 1; }
            result.signatures.push(Signature {
                kind: "class".into(), name,
                modifiers: vec![], params: None, return_type: None,
                throws: vec![], extends: None, implements: vec![],
                annotations: vec![], line: node.start_position().row + 1,
                has_docstring: has_doc,
            });
            if let Some(body) = node.child_by_field_name("body") {
                walk_ts(body, bytes, source, result, documented, total);
            }
            return;
        }
        "function_declaration" | "export_statement" => {
            // For export_statement, recurse to find inner declarations
            let mut cursor = node.walk();
            for child in node.children(&mut cursor) {
                walk_ts(child, bytes, source, result, documented, total);
            }
            return;
        }
        "method_definition" | "public_field_definition" => {
            let name = node.child_by_field_name("name")
                .map(|n| node_text(n, bytes).to_string())
                .unwrap_or_default();
            let has_doc = preceding_jsdoc(node, source);
            let kind = if node.kind() == "method_definition" { "method" } else { "field" };
            *total += 1; if has_doc { *documented += 1; }
            result.signatures.push(Signature {
                kind: kind.into(), name,
                modifiers: vec![], params: None, return_type: None,
                throws: vec![], extends: None, implements: vec![],
                annotations: vec![], line: node.start_position().row + 1,
                has_docstring: has_doc,
            });
        }
        _ => {
            let mut cursor = node.walk();
            for child in node.children(&mut cursor) {
                walk_ts(child, bytes, source, result, documented, total);
            }
        }
    }
}
```

```rust
                    kind: kind.into(), name, modifiers: vec!["pub".into()],

                    extends: None, implements: vec![], annotations: vec![],
                    line: node.start_position().row + 1, has_docstring: has_doc,

        _ => {
```

The resolver maps `raw_imports` → relative file paths within the project. It has access to all discovered file paths.

- [ ] **Step 1: Write failing tests**

Create `src/resolver.rs`:
```rust
use crate::models::Language;
use std::collections::HashMap;
use std::path::Path;

/// Given a raw import string and the set of known project files,
/// return the relative path of the matched file (if any).
pub struct Resolver {
    /// Map from various lookup keys → relative path
    /// Keyed differently per language
    file_index: HashMap<String, String>,
}

impl Resolver {
    pub fn new(lang: &Language, rel_paths: &[String], project_root: &Path, go_module: Option<&str>) -> Self {
        let mut file_index = HashMap::new();
        match lang {
            Language::Java => {
                // "com.example.model.User" → "src/main/java/com/example/model/User.java"
                for p in rel_paths {
                    if let Some(key) = java_path_to_fqn(p) {
                        file_index.insert(key, p.clone());
                    }
                }
            }
            Language::Python => {
                // "myapp.models" → "myapp/models.py" or "myapp/models/__init__.py"
                for p in rel_paths {
                    let key = python_path_to_module(p);
                    file_index.insert(key, p.clone());
                }
            }
            Language::TypeScript | Language::JavaScript => {
                // Relative imports: "./services/user.service" → resolve relative to importer
                // Index by rel_path without extension
                for p in rel_paths {
                    let without_ext = strip_extension(p);
                    file_index.insert(without_ext.clone(), p.clone());
                    file_index.insert(p.clone(), p.clone());
                }
            }
            Language::Go => {
                // "github.com/myorg/myapp/models" with module "github.com/myorg/myapp"
                // → "models/" directory files
                if let Some(module) = go_module {
                    for p in rel_paths {
                        let pkg_path = go_pkg_path(p);
                        let import_path = format!("{}/{}", module.trim_end_matches('/'), pkg_path.trim_start_matches('/'));
                        file_index.insert(import_path, p.clone());
                    }
                }
            }
            Language::Rust => {
                // "crate::models::User" → "src/models/user.rs" or "src/models.rs"
                for p in rel_paths {
                    let key = rust_path_to_module(p);
                    file_index.insert(key, p.clone());
                }
            }
            Language::CSharp => {
                // "MyApp.Models" → "Models/*.cs" files in that namespace
                for p in rel_paths {
                    if let Some(key) = csharp_path_to_namespace(p) {
                        file_index.insert(key, p.clone());
                    }
                }
            }
            Language::Cpp => {
                // Quoted includes: "mylib/foo.h" → "mylib/foo.h" (relative path)
                for p in rel_paths {
                    file_index.insert(p.clone(), p.clone());
                    // Also index by basename
                    if let Some(name) = Path::new(p).file_name().and_then(|n| n.to_str()) {
                        file_index.entry(name.to_string()).or_insert(p.clone());
                    }
                }
            }
            Language::Ruby => {
                // require_relative 'models/user' → 'models/user.rb'
                for p in rel_paths {
                    let without_ext = strip_extension(p);
                    file_index.insert(without_ext, p.clone());
                }
            }
        }
        Self { file_index }
    }

    /// Resolve a raw import (for TS/JS, also needs the importer's dir)
    pub fn resolve(&self, raw: &str, importer_dir: Option<&str>) -> Option<String> {
        // Direct lookup
        if let Some(v) = self.file_index.get(raw) {
            return Some(v.clone());
        }
        // For relative TS/JS imports, join with importer dir and normalize
        if let Some(dir) = importer_dir {
            if raw.starts_with("./") || raw.starts_with("../") {
                let joined = Path::new(dir).join(raw);
                let normalized = normalize_path(&joined.to_string_lossy());
                if let Some(v) = self.file_index.get(&normalized) {
                    return Some(v.clone());
                }
            }
        }
        None
    }
}

fn strip_extension(p: &str) -> String {
    let path = Path::new(p);
    if let Some(stem) = path.file_stem().and_then(|s| s.to_str()) {
        if let Some(parent) = path.parent().and_then(|p| p.to_str()) {
            return if parent.is_empty() { stem.to_string() } else { format!("{}/{}", parent, stem) };
        }
    }
    p.to_string()
}

fn normalize_path(p: &str) -> String {
    // Collapse .. and . components
    let mut parts: Vec<&str> = vec![];
    for component in p.split('/') {
        match component {
            ".." => { parts.pop(); }
            "." | "" => {}
            c => parts.push(c),
        }
    }
    parts.join("/")
}

fn java_path_to_fqn(rel_path: &str) -> Option<String> {
    // "src/main/java/com/example/model/User.java" → "com.example.model.User"
    let path = Path::new(rel_path);
    if path.extension()?.to_str()? != "java" { return None; }
    let s = rel_path.replace('/', ".");
    let s = s.trim_end_matches(".java");
    // Strip common prefixes
    for prefix in &["src.main.java.", "src."] {
        if let Some(rest) = s.strip_prefix(prefix) {
            return Some(rest.to_string());
        }
    }
    Some(s.to_string())
}

fn python_path_to_module(rel_path: &str) -> String {
    let s = rel_path.trim_end_matches("/__init__.py").trim_end_matches(".py");
    s.replace('/', ".")
}

fn rust_path_to_module(rel_path: &str) -> String {
    let s = rel_path.trim_start_matches("src/").trim_end_matches("/mod.rs").trim_end_matches(".rs");
    format!("crate::{}", s.replace('/', "::"))
}
```

```rust
use std::collections::{HashMap, HashSet, VecDeque};

/// Directed graph where edge A → B means "A depends on B" (A imports B).
pub struct DepGraph {
    /// nodes: set of all node IDs
    pub nodes: HashSet<String>,
    /// edges: adjacency list (A → set of B)
    pub edges: HashMap<String, HashSet<String>>,
}

impl DepGraph {
    pub fn new() -> Self {
        Self { nodes: HashSet::new(), edges: HashMap::new() }
    }

    pub fn add_node(&mut self, id: &str) {
        self.nodes.insert(id.to_string());
        self.edges.entry(id.to_string()).or_default();
    }

    /// Add edge: from depends on to
    pub fn add_edge(&mut self, from: &str, to: &str) {
        self.edges.entry(from.to_string()).or_default().insert(to.to_string());
        self.edges.entry(to.to_string()).or_default(); // ensure to exists
    }

    /// Topological sort (Kahn's algorithm).
    /// Returns (order, cycle_pairs) where cycle_pairs are edges that were
    /// removed to break cycles. Each node in the returned order appears
    /// after all its dependencies.
    pub fn topo_sort(&self) -> (Vec<String>, Vec<(String, String)>) {
        let mut in_degree: HashMap<String, usize> = self.nodes.iter().map(|n| (n.clone(), 0)).collect();
        let mut adj = self.edges.clone();

        for (_, deps) in &adj {
            for dep in deps {
                *in_degree.entry(dep.clone()).or_insert(0) += 1;
            }
        }

        let mut queue: VecDeque<String> = in_degree.iter()
            .filter(|(_, &deg)| deg == 0)
            .map(|(n, _)| n.clone())
            .collect();
        // Sort for determinism
        let mut queue_vec: Vec<String> = queue.drain(..).collect();
        queue_vec.sort();
        queue.extend(queue_vec);

        let mut order = vec![];
        let mut cycle_pairs = vec![];
        let mut visited = HashSet::new();

        while order.len() < self.nodes.len() {
            if let Some(node) = queue.pop_front() {
                order.push(node.clone());
                visited.insert(node.clone());
                let mut deps: Vec<String> = adj.get(&node).cloned().unwrap_or_default().into_iter().collect();
                deps.sort();
                for dep in deps {
                    let deg = in_degree.entry(dep.clone()).or_insert(0);
                    *deg -= 1;
                    if *deg == 0 {
                        queue.push_back(dep);
                    }
                }
            } else {
                // Cycle detected: find the unvisited node with the most shared dependencies
                // Simple heuristic: pick the unvisited node with highest in-degree and remove one edge
                let unvisited: Vec<&String> = self.nodes.iter()
                    .filter(|n| !visited.contains(*n) && !queue.iter().any(|q| q == *n))
                    .collect();
                if unvisited.is_empty() { break; }

                // Break cycle: remove edge from the node with most shared imports
                let breaker = unvisited[0].clone();
                let deps: Vec<String> = adj.get(&breaker).cloned().unwrap_or_default()
                    .into_iter()
                    .filter(|d| !visited.contains(d))
                    .collect();
                if let Some(target) = deps.into_iter().next() {
                    cycle_pairs.push((breaker.clone(), target.clone()));
                    adj.entry(breaker.clone()).and_modify(|s| { s.remove(&target); });
                    let deg = in_degree.entry(breaker.clone()).or_insert(1);
                    if *deg > 0 { *deg -= 1; }
                    if *in_degree.get(&breaker).unwrap_or(&0) == 0 {
                        queue.push_back(breaker);
                    }
                } else {
                    // No outgoing edge to break — add to order directly
                    order.push(breaker.clone());
                    visited.insert(breaker);
                }
            }
        }

        (order, cycle_pairs)
    }
}
```

```rust
use crate::generated::is_generated;
use crate::graph::DepGraph;
use crate::models::{CacheFile, FileEntry, Language, Stats};
use crate::parsers::get_parser;
use crate::resolver::Resolver;
use crate::walker::{walk_project, WalkConfig};
use chrono::Utc;
use rayon::prelude::*;
use std::collections::HashMap;
use std::path::{Path, PathBuf};

pub struct ScanConfig {
    pub forced_lang: Option<Language>,
    pub include_globs: Vec<String>,
    pub exclude_globs: Vec<String>,
    pub min_coverage: f64,
    pub cache_dir: Option<PathBuf>,
    pub verbose: bool,
}

pub struct ScanResult {
    pub cache_path: PathBuf,
    pub stats: Stats,
    pub detected_languages: Vec<String>,
    pub order: Vec<String>,
    pub project_root: String,
}

pub fn scan(root: &Path, cfg: &ScanConfig) -> anyhow::Result<ScanResult> {
    let root = root.canonicalize()?;
    let walk_cfg = WalkConfig {
        forced_lang: cfg.forced_lang.clone(),
        include_globs: cfg.include_globs.clone(),
        exclude_globs: cfg.exclude_globs.clone(),
    };

    let discovered = walk_project(&root, &walk_cfg)?;
    if cfg.verbose {
        eprintln!("[codeskel] Discovered {} files", discovered.len());
    }

    // Collect all relative paths per language (for resolver)
    let mut paths_by_lang: HashMap<Language, Vec<String>> = HashMap::new();
    for f in &discovered {
        paths_by_lang.entry(f.language.clone()).or_default().push(f.rel_path.clone());
    }

    // Read go.mod if present
    let go_module = read_go_module(&root);

    // Parse all files in parallel
    let parsed: Vec<_> = discovered.par_iter().map(|f| {
        let content = match std::fs::read_to_string(&f.abs_path) {
            Ok(c) => c,
            Err(e) => {
                eprintln!("[codeskel] Warning: cannot read {}: {}", f.rel_path, e);
                return (f, None, None);
            }
        };
        let generated = is_generated(&f.rel_path, &content);
        let parser = get_parser(&f.language);
        let parse_result = parser.parse(&content);
        (f, Some(content), Some((generated, parse_result)))
    }).collect();

    // Build resolver per language
    let resolvers: HashMap<Language, Resolver> = paths_by_lang.iter().map(|(lang, paths)| {
        let r = Resolver::new(lang, paths, &root, go_module.as_deref());
        (lang.clone(), r)
    }).collect();

    // Build file entries and dependency graph
    let mut graph = DepGraph::new();
    let mut file_entries: HashMap<String, FileEntry> = HashMap::new();
    let mut lang_set: std::collections::HashSet<String> = std::collections::HashSet::new();
    let mut skipped_covered = 0usize;
    let mut skipped_generated = 0usize;

    for (f, _content, parse_opt) in &parsed {
        graph.add_node(&f.rel_path);
        lang_set.insert(f.language.as_str().to_string());

        let (skip, skip_reason, coverage, signatures, raw_imports, package) = match parse_opt {
            None => (true, Some("unreadable".to_string()), 0.0, vec![], vec![], None),
            Some((true, _pr)) => {
                skipped_generated += 1;
                (true, Some("generated".to_string()), 0.0, vec![], vec![], None)
            }
            Some((false, pr)) => {
                let skip = pr.coverage >= cfg.min_coverage;
                if skip { skipped_covered += 1; }
                let reason = if skip { Some("sufficient_coverage".to_string()) } else { None };
                (skip, reason, pr.coverage, pr.signatures.clone(), pr.raw_imports.clone(), pr.package.clone())
            }
        };

        // Resolve imports to internal file paths
        let resolver = resolvers.get(&f.language);
        let importer_dir = Path::new(&f.rel_path).parent().and_then(|p| p.to_str()).map(|s| s.to_string());
        let internal_imports: Vec<String> = raw_imports.iter()
            .filter_map(|raw| resolver.and_then(|r| r.resolve(raw, importer_dir.as_deref())))
            .collect();

        // Add graph edges
        for dep in &internal_imports {
            graph.add_edge(&f.rel_path, dep);
        }

        file_entries.insert(f.rel_path.clone(), FileEntry {
            path: f.rel_path.clone(),
            language: f.language.as_str().to_string(),
            package,
            comment_coverage: coverage,
            skip,
            skip_reason,
            cycle_warning: false,
            internal_imports,
            signatures,
        });
    }

    // Topological sort
    let (full_order, cycle_pairs) = graph.topo_sort();

    // Mark cycle participants
    for (a, b) in &cycle_pairs {
        eprintln!("[codeskel] Warning: cycle detected between {} and {}", a, b);
        if let Some(e) = file_entries.get_mut(a) { e.cycle_warning = true; }
        if let Some(e) = file_entries.get_mut(b) { e.cycle_warning = true; }
    }

    // Filter to non-skipped order
    let order: Vec<String> = full_order.into_iter()
        .filter(|p| file_entries.get(p).map(|e| !e.skip).unwrap_or(false))
        .collect();

    let to_comment = order.len();
    let total_files = file_entries.len();

    let mut detected_languages: Vec<String> = lang_set.into_iter().collect();
    detected_languages.sort();

    let stats = Stats { total_files, skipped_covered, skipped_generated, to_comment };

    let cache_dir = cfg.cache_dir.clone()
        .unwrap_or_else(|| root.join(".codeskel"));

    let cache = CacheFile {
        version: 1,
        scanned_at: Utc::now().to_rfc3339(),
        project_root: root.to_string_lossy().into_owned(),
        detected_languages: detected_languages.clone(),
        stats: stats.clone(),
        order: order.clone(),
        files: file_entries,
    };

    write_cache(&cache_dir, &cache)?;
    let cache_path = cache_dir.join("cache.json");

    Ok(ScanResult { cache_path, stats, detected_languages, order, project_root: root.to_string_lossy().into_owned() })
}

fn read_go_module(root: &Path) -> Option<String> {
    let gomod = root.join("go.mod");
    let content = std::fs::read_to_string(gomod).ok()?;
    content.lines()
        .find(|l| l.starts_with("module "))
        .map(|l| l.trim_start_matches("module ").trim().to_string())
}
```

```rust
        let content = match std::fs::read_to_string(&abs) {
            Ok(c) => c,
            Err(e) => {
                eprintln!("[codeskel] Warning: cannot read {}: {}", rel_str, e);
                continue;
            }
        };

        let lang = detect_language(&abs)
            .ok_or_else(|| anyhow::anyhow!("Cannot detect language for {}", rel_str))?;

        let generated = is_generated(&rel_str, &content);
        let parser = get_parser(&lang);
        let pr = parser.parse(&content);

        if let Some(entry) = cache.files.get_mut(&rel_str) {
            entry.comment_coverage = pr.coverage;
            entry.signatures = pr.signatures;
            if generated {
                entry.skip = true;
                entry.skip_reason = Some("generated".to_string());
            } else {
                entry.skip = false;
                entry.skip_reason = None;
            }
            entry.scanned_at = Utc::now().to_rfc3339();
        }
    }

    // Recompute stats
    let total = cache.files.len();
    let skipped_covered = cache.files.values().filter(|e| e.skip_reason.as_deref() == Some("sufficient_coverage")).count();
    let skipped_generated = cache.files.values().filter(|e| e.skip_reason.as_deref() == Some("generated")).count();
    let to_comment = cache.order.iter().filter(|p| cache.files.get(*p).map(|e| !e.skip).unwrap_or(false)).count();
    cache.stats = crate::models::Stats { total_files: total, skipped_covered, skipped_generated, to_comment };

    write_cache(cache_dir, &cache)?;
    eprintln!("[codeskel] Rescanned {} file(s)", args.file_paths.len());
    Ok(())
}
```

## Implementation Notes

### Tree-sitter Field Names
The field names used in this plan (`name`, `body`, `parameters`, `type`, `superclass`, `interfaces`, `declarator`, `source`, `module_name`, `path`) are based on the standard grammars, but **must be verified** against the actual compiled grammar. The fastest way:

```rust
// Debug helper: print AST to stderr
fn debug_tree(source: &str, lang: tree_sitter::Language) {
    let mut parser = tree_sitter::Parser::new();
    parser.set_language(lang).unwrap();
    let tree = parser.parse(source, None).unwrap();
    eprintln!("{}", tree.root_node().to_sexp());
}
```

### Grammar Version Compatibility
If `tree-sitter = "0.22"` conflicts with grammar crates at `"0.21"`, try:
- Pinning `tree-sitter = "0.20"` and all grammars to `"0.20"`
- Or using `tree-sitter = "0.23"` if released

Check crates.io for latest compatible sets.

### rayon + tree-sitter
`tree_sitter::Parser` is not `Send`, so parsers must be created fresh per thread. The `get_parser()` function returns a new `Box<dyn LanguageParser>` each time, and each `parse()` call creates a fresh `tree_sitter::Parser` — this is correct.

### Python `_private` Methods
The PRD says "public/protected methods" are documentable. For Python, skip methods starting with `_` except `__init__` (which is documentable as a constructor).
